# Estrutura basica

```html

Descrições

<!DOCTYPE html>: Define o tipo de documento como HTML.
<html>: Define o início do documento HTML.
<head>: Contém informações sobre a página, como título e metadados.
<title>: Define o título da página.
<meta>: Fornece metadados sobre a página, como charset e palavras-chave.
<body>: Contém o conteúdo principal da página.
</html>: Define o fim do documento HTML.

Exemplo:

<!DOCTYPE html> 
<html> 
    <head> 
        <title> Aula 1</title>
        <meta>
    </head>
    <body>
        <p>Olá</p>
    </body>
</html>
```


# Elementos de texto

```html

Descições

<p>: Define um parágrafo de texto.
<h1> a <h6>: Definem títulos de diferentes níveis.
<br>: Insere uma quebra de linha.
<hr>: Insere uma linha horizontal.

Exemplo:

<h1>Olá mundo</h1>
<hr>
<p>Este é um pequeno exemplo daquilo que vocês podem fazer com a linguagem de marcação HTML</p>
<br>
```

# Listas

```html
Descrições

<ul>: Define uma lista não ordenada.
<ol>: Define uma lista ordenada.
<li>: Define um item de lista.

Exemplo:

<p>Lista de Nomes não ordenada</p>
<ul>
    <li>Efrain</li>
    <li>Aurora</li>
</ul>

<p>Lista de Nomes ordenada</p>
<ol>
    <li>Efrain</li>
    <li>Aurora</li>
</ol>
```


# Link
```html

Descrição

<a>: Define um link para outro recurso.
href: Define o destino do link.

Exemplo

<a href="pagina-seguinte.html">Vai para outra pagína</a>
```

# Imagem
```html

Descrição

<img>: Insere uma imagem na página.
src: Define o caminho para a imagem.
alt: Define um texto alternativo para a imagem.

Exemplo

<img src="foto.jpg" alt="Carro">
```

# Formulário
```html

Descrição

<form>: Define um formulário.
Action: Define o que para onde sera enviado
Method: Define o forma como os dados serao enviados


Exemplo

<form action="" method="get"></form>
```

# Tabelas
```html

Descrição

<table>: Define uma tabela.
<tr>: Define uma linha de tabela.
<td>: Define uma célula de tabela.


Exemplo

<table>
    <tr>
        <td>Nome</td>
        <td>Número</td>
    </tr>
    <tr>
        <td>Ngori</td>
        <td>123</td>
    </tr>
</table>
```
